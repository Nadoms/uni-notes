# W1 - NLU Tasks and Applications
Common sequence classification tasks include:
- Sentiment analysis
- Emotion recognition
- Hate speech detection

Opinions can be characterised by $(o, a, so, h, t)$:
$o$ - Target entity, e.g. a product or topic
$a$ - Attribute of the entity, e.g. battery life
$so$ - Sentiment orientation of the opinion
$h$ - The opinion holder
$t$ - Timestamp of the opinion
**Explicit opinions** state the type of emotion felt, **implicit opinions** can have underlying attitude.

Pairwise sequence classification tasks:
- Natural language inference
- Paraphrase identification
- Entity linking
- Evidence retrieval
- Automated verification

Span-based classification tasks:
- Named entity recognition
- Semantic role labelling
- Aspect-based sentiment analysis
- Argument component identification
These are tasks which scan the entire text.

Span-based relation classification tasks:
- Relation extraction
- Coreference resolution
- Argument relation classification
- Event participant identification

**Digital humanities** transforms humanities research projects with statistical analysis, search and retrieval, data visualisation, etc.
